THE WEB DESIGN WORKSHOP
Every business either has a website - or needs a website. Knowing the basics of designing for the web helps you understand what is possible and can save you money.

Perhaps you want to create a stylish web presence for yourself to showcase your achievements or a portfolio of your work.

Sure, there are a lot of web applications out there that will help you do this but if you want to individualise it with your own style, you need to know HTML and CSS.

HTML and CSS are the building blocks of the web and a great entry point into learning more about creating technology.

WHAT CAN I EXPECT?
As a class, we will take a website all the way from mock-ups and prototyping, to deploying and hosting it on the web. You will also have the opportunity to build your own website from scratch with the help and guidance of our teachers.

We will teach you the latest tips and tricks and show you where to find a stack of great free tools to help you build your website quickly and efficiently.

WHAT WILL I LEARN?
The web design process
- The inner workings of the web
- How to design and plan a website
- Using wireframes for prototyping
- How to set up the hosting for your website

Technologies and tools
- HTML5 and CSS3
- Tools to style your site professionally
- Optimising your site for search engines
- Using Google Analytics to see where your visitors come from
